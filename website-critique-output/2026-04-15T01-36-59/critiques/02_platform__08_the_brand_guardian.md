## Off-Brand Elements
*   **Generic Semantic Colors in UI Mockup**
    *   *What it is:* In the "Fix it" section mockup, there is a pill labeled `✓ APPROVED - 06:31` that uses a generic emerald/stark green.
    *   *What it should be:* Per brand guidelines, semantic success/warning scales should utilize Swoop's amber/orange-deep variations, or a specifically approved muted brand tone, rather than defaulting to generic UI green. 
    *   *Where it appears:* Halfway down the page, in the "The right action. The right person." mockup block.
*   **Inconsistent CTA Directional Arrows**
    *   *What it is:* The top navigation button reads "Book a Walkthrough" while the hero and subsequent buttons read "Book a 30-Min Walkthrough →".
    *   *What it should be:* The directional right arrow (`→`) is a distinct Swoop action pattern and should be applied consistently to all primary conversion buttons, including the top nav.
    *   *Where it appears:* Top navigation bar.
*   **Clunky Full-Width Button Block**
    *   *What it is:* The CTA "Every objection answered. Book the walkthrough →" is housed in a massive, sharp-cornered, full-width orange block.
    *   *What it should be:* Primary CTAs should consistently use the rounded pill/button shape (16-20px border radius) seen elsewhere on the page to maintain component fidelity. If a full-width section is needed, it should be a colored band containing a standard pill button, not a button stretched to fit a container.
    *   *Where it appears:* Just beneath the "Why not just..." FAQ comparison cards.
