## Off-Brand Elements
**1. Pricing figures missing JetBrains Mono (Typography Fidelity)**
*   *What it is:* The main tier prices in the pricing section (`$0/mo`, `$499/mo`, `$1,499/mo`) are currently set in a sans-serif font (Plus Jakarta Sans).
*   *What it should be:* Per the brand guidelines, all financial figures and numerical data should use JetBrains Mono. This must be updated to reinforce the quantitative, data-driven nature of the platform.
*   *Where it appears:* The three pricing cards in the "Start at zero" section.

**2. Inconsistent "Book a Walkthrough" Button Styling (Pattern Consistency)**
*   *What it is:* In the "Your members feel it" section, the standalone "Book a Walkthrough" CTA uses a secondary outline style (dark border, transparent fill). 
*   *What it should be:* Standalone section CTAs driving to the primary conversion goal should retain the primary orange button style to train the user's eye. 
*   *Where it appears:* Bottom of the cream "Member Experience" section.

**3. Inconsistent CTA Arrow Usage (Pattern Consistency)**
*   *What it is:* Some primary CTAs use a right arrow (`Book the 30-Minute Walkthrough →`, `Test Swoop on your data →`), while others do not (`Book a Walkthrough` in the nav and hero).
*   *What it should be:* The button component should have a strict rule regarding directional arrows (e.g., only use arrows for inline text links, or consistently use them on all large section-ending CTAs).
*   *Where it appears:* Across various sections of the page.
