## Off-Brand Elements
*   **Uncompiled Arrows / Double Hyphens in CTAs:**
    *   *What it is:* The bottom dark section CTA reads "Book a 30-min Walkthrough --" and the footer reads "Talk to a GM who's using it -- Book a Walkthrough". 
    *   *What it should be:* These should use the proper right arrow character (`→`) or an em-dash (`—`) to match the buttons higher up on the page. Double hyphens look like draft copy or a code slip.
    *   *Where it appears:* Bottom CTA band and Footer text link.
*   **Minor Capitalization Inconsistencies in CTAs:**
    *   *What it is:* The pricing card CTA uses Title Case ("Book a 30-**Min** Walkthrough →"), while the bottom banner CTA uses lowercase for the abbreviation ("Book a 30-**min** Walkthrough --").
    *   *What it should be:* Pick one casing standard for button components and enforce it globally.
    *   *Where it appears:* Middle pricing card vs. Bottom CTA band.
*   **Missing Arrow on Outline Button:**
    *   *What it is:* The "Get Free Daily Alerts" secondary button on the $0/mo tier lacks an arrow, while the "See the Full Platform →" button on the $1,499 tier includes one.
    *   *What it should be:* Button patterns should be consistent; if primary/secondary outbound links use arrows, apply them uniformly.
